Para fazer uma lista ordenada coloca a tag ol e dentro dela cada item da lista fica dentro da tag li, a partir da versão HTML5 o barra li (/li) se tornou opçional

Você pode colocar dentro da tag ol o type para escolher como vai aparecer na frente da lista : "1" fica numerado, "i" fica em romanos "a" fica em ordem alfabética
